# FUTURE_ML_01
# Sales & Demand Forecasting

## Project Overview

This project predicts future sales using historical retail sales data.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Machine Learning Model

- Random Forest Regressor

## Features Engineered

- Year
- Month
- Day
- DayOfWeek
- Lag1
- Lag7
- Rolling7

## Evaluation Metrics

- MAE: 1730.59
- RMSE: 2323.24

## Business Value

The forecast helps businesses:

- Manage inventory
- Improve staffing plans
- Support financial forecasting
- Reduce stock shortages
